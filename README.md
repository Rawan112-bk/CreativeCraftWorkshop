# 🎨 Creative Craft Workshop Website

## 📌 Project Overview

The **Creative Craft Workshop** website is a front-end web development project designed to provide users with an interactive platform to explore workshops, register for events, and contact the organizers.

The website focuses on clean design, user-friendly navigation, and proper form validation using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

### 🏠 Home Page

* Attractive hero section
* Simple navigation bar
* Introduction to the workshop concept

### 🛍️ Workshops Page

* Displays available workshops:

  * Candle Making
  * Pottery Workshop
  * Painting Workshop
* Each workshop includes a **"Register Now"** button
* Clicking a button redirects to the registration page with selected workshop

### 📝 Registration Page

* Auto-fills selected workshop from workshops page
* Dynamic page title (e.g., *Candle Making Booking*)
* Clean and responsive form layout
* Fields include:

  * Full Name
  * Email
  * Mobile Number
  * Gender
  * Date & Time
  * Notes
* JavaScript validation for all inputs
* Success and error messages displayed clearly

### 📩 Contact Page

* Contact form with user-friendly layout
* Fields include:

  * First Name & Last Name
  * Email
  * Mobile Number
  * Date of Birth
  * Gender
  * Language selection
  * Message
* Front-end validation using JavaScript
* Error messages shown in red
* Success message shown in green

---

## 🧠 Technologies Used

* **HTML5** → Structure of the website
* **CSS3** → Styling and layout (responsive design)
* **JavaScript (Vanilla JS)** → Form validation and interactivity

---

## ✅ Validation Features

* Checks for:

  * Empty fields
  * Correct email format
  * Mobile number format (10 digits)
  * Name contains letters only
  * Minimum message length
* Displays:

  * ❌ Error messages (red)
  * ✅ Success messages (green)
